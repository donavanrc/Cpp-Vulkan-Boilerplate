# C++ Vulkan Boilerplate

A basic Vulkan application template written in C++.

## Prerequisites

| Software | Platform |
| --- | --- |
| CMake 3.26+ | Windows / Linux |
| Python 3.1+ | Windows / Linux |
| GCC 11+ | Windows / Linux |
| Vulkan SDK 1.2+ | Windows / Linux |

## Project build

```bash
mkdir build
cd build
cmake .. -G "Visual Studio 17 2022" -A x64
```

## Project run

```bash
cmake --build build
```
